### Описание
Реализована функция автоматического распределения обращений по операторам, ручное распределение,
проверка нераспределенных обращений.
В crud функциях бизнес логики (distribution) расписана логика.
Помимо этого, реализованы create, read эндпоинты.
Бизнес логика мне показалась непростой и местами неправильно сформулированной и было много попыток проектирование
моделей заново, потратил много времени.
Бизнес логику помог понять AI-агент DeepSeek V3.2.

### Запуск с помощью Docker Compose
1. Перейдите в пустую папку и клонируйте репозиторий
 - ```git clone https://github.com/Stepan1771/lead_processing_test_task.git```
2. Создайте виртуальное окружение и загрузите зависимости
3. Перейдите в папку "app"
 - ```cd app```
4. Запустите
 - ```uvicorn main:main_app```
